## Installation
1. Clone the repository: 
    ```
    git clone https://github.com/SamirYousri/NewsApp.git
    ```
2. Navigate to the project directory:
    ```
    cd NewsApp
    ```
3. Install dependencies:
    ```
    flutter pub get
    ```
4. Run the app:
    ```
    flutter run
    ```
